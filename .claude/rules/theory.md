# Rule: 이론 md 생성

이론 정리 md 생성 요청 시 아래 템플릿을 사용한다.
저장 위치: `theory/` 폴더

## 이론 md 템플릿

```markdown
# {알고리즘명}
> 생성일: {날짜} | Claude Code로 생성

---

## 핵심 개념

{한 줄 요약}

---

## 언제 쓰나?

| 상황 | 예시 |
|------|------|
| ... | ... |

---

## 동작 원리

{설명 + 흐름도 또는 표}

---

## Java 구현 템플릿

\`\`\`java
// 핵심 패턴 코드
\`\`\`

---

## 대표 문제 패턴

### 패턴 1: {패턴명}
\`\`\`java
// 코드
\`\`\`

---

## 시간/공간 복잡도

| 연산 | 복잡도 |
|------|--------|
| ... | O(?) |

---

## 주의사항 & 팁

- ...

---

## 연습 문제 (프로그래머스)

| 레벨 | 문제명 | 핵심 포인트 |
|------|--------|-------------|
| Lv1 | ... | ... |
| Lv2 | ... | ... |
```

## 규칙

- Java 예제 코드 반드시 포함
- 표와 흐름으로 시각화
- 프로그래머스 추천 문제 최소 3개
- 파일명 형식: `{번호}_{영문명}.md` (예: `05_bfs_dfs.md`)
