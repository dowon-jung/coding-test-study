## 생성 규칙

- 저장 위치: `theory/` 폴더
- 파일명: `{번호}_{영문명}.md` (예: `05_bfs_dfs.md`)
- 반드시 Java 예제 코드 포함
- 표와 흐름으로 시각화
- 프로그래머스 추천 문제 최소 3개
